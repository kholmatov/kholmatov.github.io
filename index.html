Image resize project

This project is helping to solve a problem with media content in a TV guide.

Our operators create illustrations in two formats vertical and horizontal in big size for tv events, then upload to our cloud storage.

Our goal is to make the image available for different platforms such as mobile/tablet/PC/TV.

Delivering pictures of different sizes through our system to over a million users was not an easy task. So that the system works 24/7 without interruption and has the ability to auto-scaling.

To solve this problem, my team and I organized a brainstorming session and came to the decision that we will use the following technologies:

  • Kubernetes - used to automate software deployment and to set up automatic scaling.
  • Docker - to create an image of our system
  • Network Load Balancer - automatically distributes your incoming traffic across multiple node
  • S3 bucket - for caching processed images
  • Python with FastAPI web framework for developing API and pillow library for manipulation with images
